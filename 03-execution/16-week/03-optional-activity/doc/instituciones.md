## Notas de implementación para el Sistema de Horarios SENA

1. **Clave primaria sugerida para Regional:** código DANE del departamento (ej. `41` = Huila).
2. **Clave primaria sugerida para Centro:** código interno SENA de 4 dígitos (ej. `9527`).
3. **Sede** es un subconjunto del centro; no todos los centros tienen múltiples sedes.  
   Ejemplo: Centro 9527 Neiva → Sede Industria, Sede Comercio, Sede Servicios.
4. **Bloque y Piso** son nivel físico-arquitectónico, se registran manualmente por cada centro.
5. **Ambiente** es la unidad mínima de gestión del módulo (código único por centro).
